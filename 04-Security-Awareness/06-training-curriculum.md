# Security Awareness — Annual Training Curriculum

**Phase:** 4 — Security Awareness & Phishing Program
**Objective:** Define the annual security awareness training curriculum for all employees
**Delivery:** LMS (Learning Management System) + Live sessions + Simulated exercises
**Frequency:** Annual baseline + quarterly reinforcement + role-specific modules

---

## Annual Curriculum Overview

| Module | Audience | Format | Duration | Frequency |
|--------|----------|--------|----------|-----------|
| New Hire Security Onboarding | All new hires | Self-paced LMS | 45 min | Once (upon hire) |
| Annual Security Awareness | All employees | Self-paced LMS | 60 min | Annual |
| Phishing Simulation | All employees | Simulated email | Ongoing | Quarterly (4 waves) |
| BEC Prevention | Finance + Execs | Live workshop | 30 min | Quarterly |
| Secure Coding | Engineering | Live workshop | 60 min | Semi-annual |
| Incident Response Basics | All employees | Self-paced LMS | 20 min | Annual |
| Role-Based Training | Per role | Varied | 30–60 min | Annual |

---

## Annual Core Module — Topics

| Topic | Description | Duration |
|-------|-------------|----------|
| Phishing & Social Engineering | Email phishing, spear phishing, BEC, vishing, smishing | 15 min |
| Physical Security | Tailgating, badge security, clean desk, visitor management | 5 min |
| Password Security | Password managers, MFA, recognizing credential harvest attempts | 10 min |
| Data Classification | How to classify and handle L1–L4 data, labeling requirements | 10 min |
| Remote Work Security | VPN use, public Wi-Fi risks, device security, screen privacy | 5 min |
| Incident Reporting | How to recognize and report security incidents, Phish Alert Button | 5 min |
| USB Drop Attacks | Never insert unknown USB devices, report found drives to IT | 3 min |
| Deepfakes & AI Threats | Awareness of AI-generated voice/video impersonation | 5 min |
| Insider Threat Awareness | Recognizing and reporting suspicious behavior by colleagues | 2 min |

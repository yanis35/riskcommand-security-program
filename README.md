# RiskCommand — Build a CISO's Entire Security Program from Day Zero

**Client:** Confidential (Series B Fintech — 200 employees, $40M funding)
**Role:** Inaugural CISO
**Mandate:** SOC 2 Type II readiness in 12 months, full security program build-out
**Budget:** $500,000
**Timeline:** 12 weeks for program design, 12 months for SOC 2 readiness

---

## Scenario

You are hired as the first CISO of a fast-growing fintech startup. The company processes sensitive financial data, maintains PCI DSS compliance obligations, and has committed to SOC 2 Type II certification within 12 months to close enterprise deals. There is no existing security program — no policies, no risk register, no incident response plan, no vendor risk management, no security awareness program. Everything must be built from scratch.

The board expects:
- A quantified risk posture within 30 days
- A complete policy framework within 60 days
- Tested BCP/DR plans within 90 days
- A functioning security awareness program with measurable improvement
- Vendor risk management for 30+ SaaS providers
- Board-ready quarterly reporting with KRIs and ROSI calculations

**Budget allocation:** $500K across people (Security Engineer + SOC analyst), tools (SIEM, EDR, CSPM, phishing platform), and external assessments (pentest, SOC 2 auditor).
